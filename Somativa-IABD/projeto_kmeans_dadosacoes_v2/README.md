
# Projeto - Agrupamento de Ações com K-Means (dados oficiais v2)

Esta versão já está ajustada para o CSV oficial `dadosacoes.csv` com as colunas:
- nome ação
- preço ação R$
- qtde cotas
- valor de mercado R$ -(Bilhões)

## Como executar

1. Crie e ative um ambiente virtual (opcional).
2. Instale as dependências:

   python -m pip install -r requirements.txt

3. Certifique-se de que `dadosacoes.csv` está na mesma pasta que `main.py`.
4. Execute:

   python main.py
